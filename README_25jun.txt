
PROYECTO: Encuesta Escolar Web con Python y Flask

FECHA DE INICIO: 25 de Junio de 2025

DESCRIPCIÓN GENERAL:
Este es un proytecto de práctica personal para aprender desarrollo web con Python y Flask.
Consiste en una aplicación web básica que permite realizar encuestas escolares simples.

OBJETIVOS DEL PROYECTO:
-Aprender a programar con Python desde Cero.
-Crear una aplicación web funcional utilizando Flask.
-Desarrollar habilidades de documentación, control de versiones y testing.
-Preaparar un proyecto que pueda incluir en mi portafolio para futuras postulaciones laborales.

AVANCES DEL DÍA
-Instalé Python y Configuré Visual Studio Code con la extención de Python.
-Instalé Flask con el comando 'pip install flask'
-Creé la estructura inicial del proyecto:
	/EncuestaApp
	  |- app.py
	  |- templates/
-Escribí un archivo app.py con una ruta simple ('/') que muestra 'Hola Mundo'.
-Probé la app localmente en http://127.0.0.1:5000

PRÓXIMOS PASOS:
